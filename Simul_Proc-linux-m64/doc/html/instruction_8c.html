void print_op ( Instruction  instr)

Impression de la variable de l'instruction.

La fonction print_op permet d'imprimer la variable de l'instruction passée en paramètre. Elle regarde de quel type est l'instruction (direct, indirect, ...) et choisi le mode d'affichage en fonction.

Paramètres
instrl'instruction dont on doit imprimer la variable
